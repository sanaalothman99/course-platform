# A to Z Automation — Frontend

Online PLC training platform built with Next.js 15.

## 🚀 Live Demo
[atozautomation.de](https://atozautomation.de)

## 🛠️ Tech Stack
- Next.js 15 (App Router)
- TypeScript
- Tailwind CSS

## ✨ Features
- 🎓 Course listing and detail pages
- 💳 Stripe payment integration
- 🔐 JWT authentication
- 📊 Student progress tracking
- 💬 Comments system
- 📱 Fully responsive design
- ⚙️ Admin dashboard
- 🎬 Preview videos
